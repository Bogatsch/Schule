Was sind Variablen?

Eine Variable ist wie eine beschriftete Box, in der du einen Wert speichern kannst. Jede Box hat einen Namen und einen Typ (welche Art von Inhalt hineinpasst).

Anweisungen

Was ist eine Anweisung?

Eine Anweisung ist ein einzelner Befehl an den Computer. Anweisungen werden nacheinander von oben nach unten ausgefuehrt – genau wie einzelne Schritte in einem Kochrezept.

Was sind Schleifen?

                        
Schleifen wiederholen Anweisungen, solange eine Bedingung wahr ist. Bei einer kopfgesteuerten Schleife (while) wird die Bedingung vor jedem Durchlauf geprueft.

Was sind Funktionen?

                        
Eine Funktion ist ein benannter Block von Anweisungen, den du immer wieder aufrufen kannst. Sie hilft dir, deinen Code uebersichtlich zu halten und Wiederholungen zu vermeiden.
